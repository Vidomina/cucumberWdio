# cucumberWdio
To setup project:
  - run command
```
npm i
```

To run UI test (Exercise 1):
  - set credentials in the credentials.js file
  - run command
  ```
  npm run test:ui
  ```
  ![image](https://user-images.githubusercontent.com/12064839/232565618-b25679e7-9841-4733-bdf3-584c9a88c0f5.png)

To run API test (Exercise 2):
  - run command
  ```
  npm run test:api
  ```
